Council of High Intelligence

18 AI personas. One difficult decision.

This Coolify deployment packages the repository as a lightweight public landing page for installation, documentation, and source access. The project itself remains an installer-driven Claude/Codex skill, not a long-running backend application.

View on GitHub Open README Download install.sh

Install

Use the repository locally, then install into Claude Code or Codex.

git clone https://github.com/0xNyk/council-of-high-intelligence.git cd council-of-high-intelligence ./install.sh --codex

What This Deploys

  • A static project landing page served by Nginx.
  • Direct access to README.md, LICENSE, and install.sh.
  • No runtime secrets, database, or persistent volume required.

Requirements For Actual Use

  • Claude Code CLI for Claude usage.
  • Codex for Codex skill usage.
  • Optional provider CLIs like Codex CLI, Gemini CLI, or Ollama for multi-provider routing.

Why This Shape

The repository does not contain a web app or API service to run continuously in Coolify. Packaging it as a static site keeps the deployment honest: it gives you a clean public project page without inventing infrastructure the repo does not need.

For interactive use, users still clone the repo and run the installer locally inside Claude Code or Codex.
